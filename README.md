# Proyecto Integrador: Plataforma Web de Gestión de Turnos Médicos
## Materia: Programación II
## Carrera: Ingeniería en Informática

¡Bienvenido! Como trabajo integrador para la materia Programación II, el desafío consistió en abordar un proyecto de desarrollo web desde una perspectiva de consultoría real. El objetivo principal fue diseñar un sitio web adaptado a las necesidades de un microemprendimiento o profesional independiente.

El enfoque principal del trabajo estuvo puesto en simular un entorno de consultoría profesional para una **Doctora Especialista en Tocoginecología y Obstetricia**, abordando desde la interacción inicial con el cliente hasta la presentación de un prototipo interactivo funcional listo para producción.

---

## 📋 Fases del Proyecto y Metodología

### 1. Consultoría y Relevamiento de Información
* **Entrevistas con el Cliente:** Conducción de reuniones directas con la especialista para captar la visión del negocio, identificar sus necesidades de comunicación, entender la identidad de su marca y delimitar los objetivos del sitio web. Asimismo, se logró identificar al público objetivo (pacientes nuevos y frecuentes) y relevar de manera detallada los servicios de salud prestados.
* **Recopilación de Contenido:** Extracción de la información clave, imágenes y recursos que debían estructurar y reflejar las secciones de la plataforma, buscando un esquema que agilizara el proceso de autogestión de citas médicas.

### 2. Ingeniería de Requerimientos y Diseño del Sistema
* **Flujo de Reserva de Turnos:** Diseño de una interfaz clara para la agenda de citas, implementando una lógica de discriminación según el tipo de cobertura médica seleccionada por el paciente:
  * Atención Médica Particular.
  * Atención mediante Obra Social.
* **Modelo de Base de Datos:** Análisis y estructuración de los requerimientos de almacenamiento del sistema para la gestión del consultorio. Se diseñó el esquema para recopilar y almacenar los datos esenciales de los pacientes:
  * Nombre y Apellido.
  * Teléfono Celular.
  * Correo Electrónico.
  * Domicilio.
    
### 3. Prototipado y Simulación de Entrega
* **Diseño de Interfaz (UI/UX):** Estructuración visual de las pantallas buscando una navegación fluida, estética atractiva y una experiencia de usuario óptima.
* **Simulación Funcional:** Creación de un prototipo funcional que simula con fidelidad el comportamiento del sitio web terminado, validando los flujos de navegación antes de la entrega final.

---

## 🛠️ Tecnologías y Herramientas Utilizadas

Para llevar a cabo este proyecto, implementamos y trabajamos con los siguientes recursos:

* **Maquetación y Estilos:** `HTML5` (para la estructura semántica de la página) y `CSS3` (para el diseño, paleta de colores y adaptabilidad).
* **Diseño Gráfico y Recursos Visuales:** `Adobe Photoshop` (archivos `.psd` y `.psb` para la edición y optimización de fondos e imágenes como `bgDoc.png`).
* **Control de Versiones:** `Git` y `GitHub` para el seguimiento del código y el trabajo colaborativo.
